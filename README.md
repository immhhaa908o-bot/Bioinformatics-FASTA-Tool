# Bioinformatics-FASTA-Tool
Python script for analyzing DNA sequences: base counting, GC content, codon table, transcription and translation.


This project contains Python scripts developed for analyzing DNA sequences from FASTA files.  
Functions include:
- Counting nucleotides (A, T, G, C)
- Calculating GC content
- Generating reverse complement sequences
- Codon frequency calculation
- Transcription (DNA → RNA)
- Translation (RNA → Protein)

**Language:** Python  
**Environment:** VS Code  
**Author:** Ali Abbas Hussein  
**Year:** 2025  

This tool demonstrates core bioinformatics and computational biology skills by combining sequence analysis and data processing techniques.
